Kebijakan Privasi

Terakhir diperbarui: 01 Oktober 2025

Absensi Online Siswa ("kami") berkomitmen untuk melindungi privasi Anda. Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, menyimpan, dan melindungi informasi Anda saat menggunakan aplikasi berbasis web dan PWA ini ("Layanan").

1. Informasi yang Kami Kumpulkan

Untuk mendukung fungsi manajemen sekolah yang komprehensif, kami mengumpulkan jenis informasi berikut:

  • Informasi Akun Pengguna: Saat Anda masuk menggunakan Google, kami menyimpan nama, alamat email, dan foto profil Anda untuk identifikasi, autentikasi, dan pengelolaan sesi pengguna.
  • Data Sekolah dan Wilayah: Kami menyimpan data struktur organisasi sekolah, termasuk nama sekolah, dan lokasi administratif dalam hierarki wilayah (Provinsi, Kabupaten/Kota, Kecamatan).
  • Data Siswa dan Orang Tua: Kami mengumpulkan nama siswa, kelas, dan alamat email orang tua yang diasosiasikan dengan siswa tersebut. Email orang tua digunakan khusus untuk memberikan akses ke fitur "Dasbor Orang Tua".
  • Data Absensi: Catatan kehadiran harian (Hadir, Sakit, Izin, Alpa) beserta metadata waktu pencatatan dan identitas pencatat data.

2. Penggunaan Informasi

Informasi yang dikumpulkan digunakan untuk:

  • Manajemen Operasional: Menyediakan fungsionalitas inti pencatatan kehadiran, pengelolaan kelas, dan administrasi sekolah.
  • Sinkronisasi dan Aksesibilitas: Memungkinkan sinkronisasi data antar perangkat dan mendukung mode offline (PWA) agar aplikasi tetap dapat digunakan tanpa internet.
  • Analisis dan Pelaporan: Menyajikan visualisasi grafik, laporan persentase kehadiran, dan rekapitulasi data bagi Guru, Kepala Sekolah, dan Dinas Pendidikan.
  • Analisis Kecerdasan Buatan (AI): Data kehadiran agregat diproses menggunakan teknologi Google Gemini API untuk menghasilkan wawasan otomatis, seperti deteksi dini wabah penyakit di kelas, pola ketidakhadiran berulang, atau anomali data, guna membantu pengambilan keputusan sekolah.
  • Pemantauan Orang Tua: Memberikan akses kepada orang tua siswa yang terverifikasi untuk memantau riwayat kehadiran anak mereka secara real-time.

3. Penyimpanan dan Keamanan Data

  • Database Cloud Terpusat: Seluruh data aplikasi disimpan secara aman di server database cloud (PostgreSQL) yang dikelola oleh penyedia layanan kami. Data tidak disimpan di Google Drive pribadi pengguna, melainkan di infrastruktur aplikasi terpusat untuk memungkinkan kolaborasi real-time dan analisis lintas sekolah.
  • Penyimpanan Lokal (Offline): Untuk mendukung fitur offline, salinan data terenkripsi sementara disimpan di browser perangkat Anda (IndexedDB/LocalStorage). Data ini akan disinkronkan secara aman ke server saat perangkat kembali terhubung ke internet.
  • Kontrol Akses: Kami menerapkan kontrol akses berbasis peran (Role-Based Access Control) yang ketat. Guru hanya dapat mengelola kelas yang ditugaskan, Admin Sekolah hanya dapat mengakses data sekolahnya, Dinas Pendidikan hanya dapat melihat data dalam yurisdiksinya, dan Orang Tua hanya dapat melihat data anaknya sendiri.

4. Berbagi Data dan Pihak Ketiga

Kami tidak menjual, menyewakan, atau memperdagangkan data pribadi Anda. Data mungkin diproses oleh pihak ketiga dalam konteks berikut:

  • Penyedia Layanan AI: Teks ringkasan data absensi (anonim atau pseudo-anonim) dikirim ke API Google Gemini untuk keperluan analisis pola. Google memproses data ini sesuai dengan standar privasi API enterprise mereka dan tidak menggunakannya untuk melatih model AI publik.
  • Otoritas Pendidikan: Data sekolah dan agregat kehadiran dapat diakses oleh akun tingkat Dinas Pendidikan yang menaungi wilayah sekolah tersebut untuk keperluan monitoring, evaluasi kebijakan, dan pelaporan daerah.

5. Hak Pengguna

Anda memiliki hak untuk meminta akses, koreksi, atau penghapusan data siswa atau sekolah yang Anda kelola. Hal ini dapat dilakukan melalui fitur manajemen di dalam aplikasi atau dengan menghubungi administrator sistem.

6. Perubahan Kebijakan

Kami dapat memperbarui Kebijakan Privasi ini seiring dengan penambahan fitur baru (seperti fitur AI atau integrasi regional). Perubahan akan diinformasikan melalui aplikasi. Kami menyarankan Anda meninjau halaman ini secara berkala.
