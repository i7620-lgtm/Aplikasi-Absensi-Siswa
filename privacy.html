Kebijakan Privasi

Terakhir diperbarui: 05 September 2025

Aplikasi Absensi Siswa ("kami") berkomitmen untuk melindungi privasi Anda. Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, mengungkapkan, dan menjaga informasi Anda saat Anda menggunakan aplikasi kami ("Layanan").

Informasi yang Kami Kumpulkan

Kami mengumpulkan jenis informasi berikut:

  • Informasi Profil Pengguna: Saat Anda memilih untuk masuk dengan akun Google Anda, kami mengumpulkan nama, alamat email, dan foto profil Anda. Informasi ini digunakan semata-mata untuk mengidentifikasi Anda di dalam aplikasi dan tidak dibagikan dengan pihak ketiga mana pun.
  • Data Aplikasi: Semua data yang Anda masukkan ke dalam aplikasi, termasuk nama siswa, daftar kelas, dan catatan kehadiran, akan disimpan. Data ini penting untuk fungsionalitas inti aplikasi.

Bagaimana Kami Menggunakan Informasi Anda

Kami menggunakan informasi yang kami kumpulkan untuk:

  • Menyediakan dan memelihara Layanan: Data Anda digunakan untuk mengoperasikan aplikasi, memungkinkan Anda mengelola daftar siswa dan melacak kehadiran.
  • Mengaktifkan pencadangan dan sinkronisasi data: Jika Anda masuk dengan Google, aplikasi akan membuat Google Sheet di Google Drive pribadi Anda untuk mencadangkan dan menyinkronkan data Anda. Ini memungkinkan Anda mengakses data Anda di berbagai perangkat dan memulihkannya jika Anda membersihkan data browser Anda. Aplikasi hanya mengakses file spesifik ini dan tidak mengakses file lain di Google Drive Anda.

Penyimpanan dan Keamanan Data

  • Penyimpanan Lokal: Secara default, semua data aplikasi disimpan secara lokal di perangkat Anda dalam penyimpanan browser web Anda. Data ini bersifat pribadi bagi Anda dan tidak dikirimkan ke server kami.
  • Google Drive dan Google Sheets: Saat Anda menghubungkan akun Google Anda, sebuah spreadsheet bernama "Data Absensi Siswa" akan dibuat di Google Drive Anda. Semua data siswa dan kehadiran Anda disimpan di sheet ini. Anda memiliki kontrol penuh atas file ini, termasuk kemampuan untuk melihat, mengedit, atau menghapusnya kapan saja. Kami tidak memiliki akses ke Google Drive Anda atau file ini.

Berbagi Data

Kami tidak membagikan, menjual, atau menyewakan informasi pribadi atau data aplikasi Anda kepada pihak ketiga mana pun untuk tujuan apa pun.

Hak Data Anda

  • Akses dan Koreksi: Anda dapat mengakses dan mengubah data Anda kapan saja melalui antarmuka aplikasi.
  • Penghapusan Data: Anda dapat menghapus data Anda dengan membersihkan cache dan data situs browser Anda. Jika Anda menggunakan fitur sinkronisasi Google, Anda juga dapat menghapus spreadsheet dari Google Drive Anda untuk menghapus cadangan cloud Anda secara permanen.

Privasi Anak

Layanan ini tidak ditujukan untuk digunakan oleh individu di bawah usia 13 tahun. Kami tidak dengan sengaja mengumpulkan informasi pribadi dari anak-anak di bawah 13 tahun.

Perubahan pada Kebijakan Privasi Ini

Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu. Kami akan memberitahu Anda tentang perubahan apa pun dengan memposting Kebijakan Privasi baru di dalam aplikasi. Anda disarankan untuk meninjau Kebijakan Privasi ini secara berkala untuk setiap perubahan.
